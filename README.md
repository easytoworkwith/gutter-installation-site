# Gutter Installation Website

## Before running ads

Open `src/lib/site.ts` and replace the placeholder phone number with the real network phone number:

```ts
phoneDisplay: '(888) 555-0123',
phoneHref: 'tel:+18885550123',
```

## Deploy

Upload this folder to GitHub and connect it with Vercel.

Main landing page:

`/gutter-installation`

## Google Ads notes

Use the `/gutter-installation` page as Final URL. Keep exact and phrase match keywords initially. Add campaign-level negative keywords before launch.
